Scope — CS Audience Only

This capture guide covers the CS-audience inForm Safety course (~10 minutes, product awareness + talk tracks). A separate capture list will be created for the Tech Support course when that storyboard is built.

Captures dropped from 6 to 2: the CS course doesn't do a dashboard deep-dive, so we only need a hero glimpse of Company Snapshot and a heat map to support Section 2 (Feature Highlights).

What moved out Portal navigation, BASICs close-up, Data Insights tab, and Subscriptions management were all part of the original single-audience module. They belong to the Tech Support course (pending). If you're capturing anyway, grab all six in one pass — but only the two listed below are required for the CS build.

Resolution: 1920×1080 minimum. Full-page screenshots preferred. Annotate key areas with numbered callouts after capture.

Naming convention: inForm_[number]_[description].png

Save to: Articulate POC/inForm Safety/inForm_captures/
