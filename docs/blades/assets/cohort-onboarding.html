Welcome to the cohort, Commander. Before your hauls count for anything, we need your ship talking to the fleet. Three relays and a channel key — that's the whole checklist. Work the stages in order; each ends with a confirmation signal so you know it took. If the lore gets in the way, flip SIGNAL → PLAIN up top for bare instructions.

This gets you set up to feed and read the Onyx Blades colonization board. Four stages: install EDMC, add the RavenColonial plugin, link into the build, then log into the board. Each stage has a confirmation check. Toggle SIGNAL → IN-UNIVERSE for the flavored version.

01
Deploy the Market-Connector Relay
EDMC · the trade-net beacon
~5 min

Every Blade runs a Market-Connector relay. Dock anywhere and it quietly broadcasts that station's manifest to the galactic trade net — which is how the board knows where to source the cargo your build is screaming for. No relay, and your markets are dark to the fleet. Fit it once and forget it's there.

  1. Download & install EDMC (Elite Dangerous Market Connector) from EDCD. Windows installer is the top asset on the releases page.
  2. Launch it and sign in with your Frontier account when asked (same login as the game).
  3. Leave “Send to EDDN” ON (it's the default) — that broadcast is exactly what powers the board's supplier search.
Needs: EDMC 5.0 or newer. Runs alongside the game — start it before you fly.
⬇ EDMC — Releases
✔ Confirmation signal: EDMC shows your CMDR name and current system. Dock at any station and the “EDDN” indicator blips — you're on the net.
02
Install the Colonial-Survey Uplink
RavenColonial plugin for EDMC
~5 min

The uplink is the clever bit. The instant you unload at a construction ship, it reads the depot's tally — what's required, what's aboard, how close the build is — and relays it to the cohort. That's what fills the board's manifest and keeps everyone's shopping list honest. You'll never hand-file a delivery report again.

  1. Download the latest RavenColonial EDMC plugin release (the .zip) from its GitHub.
  2. In EDMC: File → Settings → Plugins tab → click “Open”. That opens your plugins folder:
%LOCALAPPDATA%\EDMarketConnector\plugins
  1. Extract the zip into that folder so you end up with:
    plugins\Ravencolonial-EDMC\load.py
  2. Fully close and reopen EDMC (not just the window — quit it).
macOS: ~/Library/Application Support/EDMarketConnector/plugins · Linux: ~/.local/share/EDMarketConnector/plugins
⬇ RavenColonial plugin — Releases ◉ ravencolonial.com
✔ Confirmation signal: after the restart, EDMC shows a new RavenColonial panel/row (and a 🚧 button that lights up when you're docked at a construction ship).
03
Link Into the Cohort Build
claim or join the construction site
in-game

Now fly to the beacon and set down on the construction ship. The uplink wakes on contact. If a Blade already broke ground here, you'll be hailed as “not yet a member” — accept, and every ton you drop starts biting into the shared manifest. If you're the architect raising a fresh site, log the project yourself and send the coordinates back to the squad so Actual can pin it to the board.

  1. Fly to your colonization site and dock at the construction ship (the 🚧 button only works while docked there).
  2. Joining BigSkinny's build: docking at a site already tracked pops a “not a member of this project” prompt → join it. Your deliveries now pool with the cohort automatically.
  3. Starting your own: click “🚧 Create Project” → set build type + name (architect auto-fills), drop the squad Discord link → submit. The ravencolonial.com project page opens.
  4. Send the build link (the project page URL — it carries the buildId) into #elite-dangerous so it can be wired onto the board.
Co-op, how it pools: RavenColonial does the math server-side. You and BigSkinny both just run the uplink and be on the same depot — your hauls subtract off each other's remaining list. Followed fleet-carrier holds (Hazard Bedlam, Southern Draw) count too.
✔ Confirmation signal: your delivery shows on the RavenColonial project page within a dock cycle — and once wired, under your commander tab on the board.
04
Authenticate to the Cohort Board
squad channel credentials
issued by Actual

Last, your channel key. The board is a closed net — Onyx Blades eyes only. Southern Draw Actual issues your access token; key in once and the board opens: live manifest, supplier routes plotted, and the priority queue telling you where the fleet needs you next.

  1. Open the Onyx Blades board (BigSkinny will send the link — it's a private page on wickdhub.com).
  2. Sign in with the squad credentials BigSkinny issues (gated access — likely your email + a one-time code).
  3. You're in: cohort queue, per-commander builds, and the copy-to-nav supplier list.
Provisioning: this step goes live once BigSkinny stands up the board's access gate. Until then, Stages 01–03 are the whole job — they're what make your hauls count.

▸ TROUBLESHOOT — if a signal won't confirm
  • No RavenColonial panel in EDMC? Check the path is exactly plugins\Ravencolonial-EDMC\load.py (not a doubled folder), then fully quit + reopen EDMC. Needs EDMC 5.0+.
  • 🚧 button greyed out? You must be docked at the construction ship itself, not a nearby station.
  • Deliveries not counting for the cohort? Make sure you joined the project (accepted the “not a member” prompt) and you're on the same depot as BigSkinny.
  • Suppliers look stale on the board? That's market age, not a fault — hit ⟳ Refresh. Fresh data appears as Blades visit those markets with EDMC running.
  • Stuck? Ping BigSkinny in #elite-dangerous with where the signal stalled — that feedback sharpens this manual.
