⤓ BACKUP & RECOVERY

The whole namespace is a few kilobytes. Download a snapshot, drop the JSON into the repo backups/ folder, and push via GitHub Desktop — git becomes the versioned, diffable, offsite backup. A weekly scheduled task does this automatically; this button is the on-demand version.
RESTORE — load a backup JSON to recover. Merge writes only the keys in the file (safe). Replace wipes the namespace first, then restores (disaster recovery only — requires typing to confirm).
