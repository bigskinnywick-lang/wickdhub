# Outlines

Training outlines, module structures, and curriculum maps.

---

!!! info "Add your first outline"
    Save any `.md` file to this folder and add it to the `nav` section in `mkdocs.yml`. It'll show up here automatically.

## Example structure for an outline

```markdown
# Module Title

## Learning Objectives
- Objective 1
- Objective 2

## Topics Covered
1. Topic A
2. Topic B
3. Topic C

## Duration
Estimated: 45 minutes

## Notes
Any additional context or delivery notes.
```
